# ESP32 IR Sensor Servo System

## Overview

This project demonstrates servo motor control using an ESP32 microcontroller and IR sensor interfacing. The system also includes LED indication for operation monitoring.

## Components Used

* ESP32
* Servo Motor
* IR Sensor Module
* LED
* Resistor

## Features

* Servo motor control
* IR sensor interfacing
* LED indication
* PWM-based servo movement
* Embedded systems implementation

## Working Principle

The ESP32 controls the servo motor using PWM signals. The IR sensor detects object presence and interacts with the system while the LED indicates operational status.

## Technologies Used

* ESP32
* Arduino Programming
* PWM Control
* Embedded Systems
